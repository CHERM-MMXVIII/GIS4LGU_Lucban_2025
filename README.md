# GIS4LGU_Lucban_2025
Training materials, sample datasets, and project files from the Geospatial Innovation and Solutions for Local Government Units (GIS4LGU) training held in Lucban, Quezon in 2025. Includes QGIS project files, shapefiles, lecture slides, and related resources developed by the SLSU Center for Hazard and Environmental Resource Mapping (CHERM).
